# QuranTester
1. Hardcode the chapter id into the 'chapter' variable.
2. Run the file.
3. Once a verse is shown, try to guess the next verse. To reveal the answer, press any key.
4. When the correct answer is revealed, press x and enter if you got it wrong, and enter if you got it right.
5. An exce, file is exported showing verses you got right, verses you got wrong, as well as verses that didn't even show up.
6. Doing multiple runs does not overwrite data about previous runs so that you see what you keep getting wrong!
